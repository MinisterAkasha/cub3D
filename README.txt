# Cub3D

## The goal of cub3d is to make something using raycasting (like Wolfenstein 3D).

![screenshot_1](https://github.com/MinisterAkasha/cub3D/blob/main/screenshoot_2.png?raw=true)
![screenshot_2](https://github.com/MinisterAkasha/cub3D/blob/main/screenshoot_1.png?raw=true)


- WASD to move, Left/Right directional keys to rotate camera
- ESC to quit
- Textures or colors for each sides (North, South, West, East) and Sky/Floor
- Sprites
